# 자동차 경주 게임

<hr>

## 기능 명세서 
### 01. Utils 
 - 입력 
 - 랜덤 숫자 생성기
 - 출력 메시지 집합

### 02. 자동차 클래스 
 - 멤버 변수 : 이름 / 랜덤값 / 이동 거리 
 - 이동 기능 
   - 4이상 이동
 - 이동 거리 출력

### 03. 자동차 집합 클래스
 - 멤버 변수 : 자동차 리스트 / 우승자 리스트 
 - 자동차 리스트 랜덤 숫자 부여 
 - 자동차 리스트 이동 기능
 - 자동차 리스트 출력
 - 우승자 판단 : 최대 거리를 찾고 같은 값을 가지는 자동차 객체를 리스트에 삽입  
 - 우승자 출력 : 리스트에 있는 자동차 이름을 출력

### 04. 랜덤 값 생성
 - 0~9사이 값 생성

### 05. 레이싱 클래스 
 - 한번의 시간 동안 수행되는 모든 절차를 수행
